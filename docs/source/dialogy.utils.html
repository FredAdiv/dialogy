debug_logs(func: Callable[[…], Any], arg_name='debug', attr_name='debug')[source]¶

Change debug level briefly.

There are methods that need debug logs but not all of them, not all the time. This decorator checks if it is being applied on a method or a function.

  1. If function or method, it checks the kwargs to have set a debug=True (configurable via arg_name).

  2. If a method without the kwargs, then it makes a check analogous to if instance.debug == True.

  3. If either of the first two conditions pass, we change the log level to DEBUG, execute the function and change the

    log-level back to ERROR.

  4. If neither conditions pass, we run the function as it is.

This way, we restrict debug logs to only those classes, methods and functions that need it.

Parameters

  • func (Callable[.., Any]) – A function or a method

  • arg_name (str) – The name in kwargs that implies a debug flag.

  • attr_name (str) – The attribute in an instance that implies a debug flag.
